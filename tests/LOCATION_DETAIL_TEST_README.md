## 🐛 常见问题排查

### 问题 1: 页面空白
**可能原因**:
1. 后端服务未启动
2. 路由配置错误
3. 组件导入路径错误

**解决方法**:
```bash
# 1. 检查后端日志
tail -f logs/application.log

# 2. 检查前端控制台
# 打开浏览器 DevTools → Console

# 3. 验证路由配置
cat src/router/index.js | grep -A 5 "LocationDetail"
```

### 问题 2: 图片无法显示
**可能原因**:
1. imageUrl 字段为空
2. 图片服务器不可达
3. CORS 跨域问题

**解决方法**:
```sql
-- 检查数据库中的图片 URL
SELECT id, name, imageUrl FROM locations WHERE id = 1;
```

### 问题 3: 设施标签不显示
**可能原因**:
1. facilities 字段格式错误
2. JSON 解析失败

**解决方法**:
```sql
-- 检查 facilities 字段格式
SELECT id, name, facilities FROM locations WHERE id = 1;

-- 正确的格式应该是 JSON 数组字符串
-- 例如：["wifi", "parking", "cafeteria"]
```

### 问题 4: 相关推荐不显示
**可能原因**:
1. 同一校区地点数量不足
2. 过滤逻辑有误

**解决方法**:
```sql
-- 查看该校区的地点总数
SELECT COUNT(*) FROM locations WHERE campus_id = 1;

-- 至少需要 2 个地点才能显示推荐
```
